# SimpleShare 文档站点

SimpleShare 项目的官方文档站点。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 项目结构

```
├── .vitepress/          # VitePress 配置
├── api/                 # API 文档
├── guide/               # 指南文档
├── deployment/          # 部署文档
└── index.md            # 首页
```
