# 文件管理

## 概述

SimpleShare 提供了完整的文件管理功能，包括文件上传、下载、重命名、删除、文件夹嵌套等。

## 功能特性

### 1. 文件上传

- 支持单文件和多文件上传
- 支持拖拽上传
- 实时上传进度显示
- 自动文件类型识别

### 2. 文件夹管理

- 支持创建文件夹
- 支持无限级文件夹嵌套
- 支持文件夹重命名
- 支持文件夹移动

### 3. 文件操作

- **重命名**: 修改文件或文件夹名称
- **删除**: 删除文件或文件夹（支持批量删除）
- **移动**: 将文件移动到其他文件夹
- **收藏**: 标记重要文件

### 4. 文件预览

- 支持图片预览
- 支持 PDF 预览
- 支持文本文件预览

## 使用指南

### 上传文件

1. 点击"上传"按钮或拖拽文件到上传区域
2. 选择要上传的文件
3. 等待上传完成

### 创建文件夹

1. 点击"新建文件夹"按钮
2. 输入文件夹名称
3. 选择父文件夹（可选）

### 管理文件

- **重命名**: 右键点击文件，选择"重命名"
- **删除**: 右键点击文件，选择"删除"
- **移动**: 右键点击文件，选择"移动到"
- **收藏**: 点击文件旁的星标图标

## API 接口

### 获取文件列表

```http
GET /api/files/list?parentId=123&sortBy=name&order=asc
```

### 上传文件

```http
POST /api/files/upload
Content-Type: multipart/form-data

file: <file>
parentId: <folder_id>
```

### 删除文件

```http
DELETE /api/files/:id
```

### 重命名文件

```http
PATCH /api/files/:id
Content-Type: application/json

{
  "name": "新文件名"
}
```
