# 分享系统

## 概述

SimpleShare 提供了安全的文件分享功能，通过提取码实现文件的临时分享。

## 功能特性

### 1. 提取码分享

- 6 位随机提取码
- 支持设置过期时间
- 支持访问次数限制
- 支持文件夹分享（包含所有子文件）

### 2. 分享管理

- 查看所有分享记录
- 修改分享设置
- 删除分享链接
- 查看访问统计

## 使用指南

### 创建分享

1. 选择要分享的文件或文件夹
2. 点击"分享"按钮
3. 设置分享选项：
   - 过期时间（可选）
   - 最大访问次数（可选）
4. 复制提取码或分享链接

### 访问分享

1. 访问分享页面
2. 输入 6 位提取码
3. 查看或下载文件

## API 接口

### 创建分享

```http
POST /api/shares
Content-Type: application/json

{
  "fileId": 123,
  "expiresAt": 1735689600000,
  "maxAccess": 10
}
```

### 获取分享信息

```http
GET /api/shares/:shareCode
```

### 删除分享

```http
DELETE /api/shares/:id
```
